# 📊 FNP Sales Analysis (Excel)

## 📌 Project Overview
This project analyzes sales data from Ferns N Petals (FNP), a gifting platform that serves occasions like Diwali, Raksha Bandhan, Valentine’s Day, and Birthdays. The analysis is performed using Microsoft Excel to uncover insights related to sales trends, customer behavior, and product performance.

---

## 🎯 Objective
- Analyze overall sales performance and revenue trends  
- Identify top-performing products and categories  
- Understand customer spending behavior  
- Evaluate delivery time and order patterns  

---

## 🛠️ Tools & Techniques
- Microsoft Excel  
- Pivot Tables & Pivot Charts  
- Data Cleaning & Transformation  
- Excel Functions (VLOOKUP, XLOOKUP, IF, SUMIF, COUNTIF)  
- Dashboard Creation  

---

## 📂 Dataset
- 1000+ sales records including orders, customers, and products  
- Fields include Order Date, Delivery Date, Revenue, Occasion, Category, and City  

---

## ⚙️ Steps Performed
1. Cleaned and structured raw data for analysis  
2. Created Pivot Tables to summarize key metrics  
3. Built calculated fields for KPIs like revenue and average customer spend  
4. Designed an interactive dashboard using charts and slicers  

---

## 📊 Key KPIs
- Total Orders: 1000  
- Total Revenue: ₹35,20,984  
- Average Order Delivery Time: 5.53 days  
- Average Customer Spend: ₹3,520.98  

---

## 📈 Key Insights
- Revenue peaks during **Anniversary and Raksha Bandhan occasions**  
- **Colors and Soft Toys** categories contribute highest revenue  
- Sales show clear **seasonal trends across months**  
- Top cities account for a significant portion of total orders  

---

## 📊 Dashboard Features
- Revenue by Occasion, Category, Month, and Hour  
- Top Products by Revenue  
- Top Cities by Orders  
- Interactive filters using slicers  

---

## 🚀 Business Impact
- Improved visibility into sales and customer trends  
- Enabled data-driven decision-making  
- Reduced dependency on manual reporting
