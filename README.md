# 📊 FNP Sales Analysis Dashboard

## 📌 Project Overview
This project analyzes sales data from Ferns N Petals (FNP), a gifting platform specializing in occasions like Diwali, Raksha Bandhan, Valentine’s Day, and Birthdays. The goal is to uncover insights related to sales trends, customer behavior, and product performance using an interactive Power BI dashboard.

---

## 🎯 Business Objective
- Analyze overall revenue and sales performance  
- Identify top-performing products and categories  
- Understand customer spending behavior  
- Evaluate delivery performance and order trends  
- Provide insights to improve sales strategy and customer satisfaction  

---

## 🛠️ Tools & Technologies
- Power BI  
- SQL  
- Microsoft Excel  
- Power Query (ETL)  
- DAX (Data Analysis Expressions)  

---

## 📂 Dataset
- Orders, Customers, and Products data  
- Covers multiple occasions and cities  
- Includes attributes like Order Date, Delivery Date, Revenue, Product Category, and Customer Details  

---

## ⚙️ Steps Performed
1. Data cleaning and transformation using **Power Query**  
2. Data modeling and relationship building  
3. Created KPIs using **DAX**  
4. Built interactive dashboards with filters and slicers  

---

## 📊 Key KPIs
- Total Orders: **1000**  
- Total Revenue: **₹35,20,984**  
- Average Order Delivery Time: **5.53 days**  
- Average Customer Spend: **₹3,520.98**  

---

## 📈 Key Insights
- Peak revenue observed during **Anniversary and Raksha Bandhan occasions**  
- **Colors and Soft Toys** categories generated highest revenue  
- Certain cities contribute significantly to total orders (Top 10 cities analysis)  
- Sales show seasonal spikes across specific months  
- Customer spending varies significantly based on occasion  

---

## 📊 Dashboard Features
- Revenue by Occasion, Category, Month, and Hour  
- Top 5 Products by Revenue  
- Top 10 Cities by Orders  
- Interactive filters (Date, Occasion, Delivery Period)  
- Dynamic KPI tracking  

---

## 🚀 Business Impact
- Enabled better understanding of **seasonal demand patterns**  
- Helped identify **high-revenue products and categories**  
- Improved decision-making through **visual insights instead of raw Excel data**  
- Reduced manual reporting effort with automated dashboards
